# Modern BitWise Calculator

A sleek, functional calculator built with Python and Tkinter.

## Features
- **Real-time Clock:** Displays current time in the top-left corner.
- **Dual Themes:** Switch between Light and Dark modes.
- **Operations:**
  - **Basic:** Addition (+), Subtraction (-), Multiplication (*), Division (/).
  - **Bitwise:** AND (&), OR (|), XOR (^), NOT (~), Left Shift (<<), Right Shift (>>).
- **Interactive:** Supports both Mouse clicks and Keyboard input.
- **Dynamic Display:** Shows calculation history and current input.

## How to Run
Ensure you have Python installed, then run:
```bash
python main.py
```

## Keyboard Shortcuts
- **Numbers & Operators:** Type directly.
- **Bitwise:** `&`, `|`, `^`, `~`, `<`, `>`.
- **Equals:** `Enter`.
- **Clear:** `Escape`.
- **Backspace:** `Backspace`.
